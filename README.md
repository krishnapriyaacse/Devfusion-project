# 🎟️ EventSphere
### AI-Powered Event Management & Ticket Booking Platform

EventSphere is a complete real-world event management ecosystem where organisers can create and manage events, while attendees can discover, book, and engage in events seamlessly.

The platform combines:
- Smart ticketing
- AI-powered recommendations
- QR verification
- Live analytics
- Networking tools
- Payment integration
- Gamification
- Community engagement

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/your-username/EventSphere.git
cd EventSphere
```

---

# 📦 Install Dependencies

## Frontend

```bash
cd client
npm install
```

## Backend

```bash
cd server
npm install
```

---

# 🔑 Environment Variables

## Frontend `.env`

```env
VITE_API_URL=http://localhost:5000
VITE_RAZORPAY_KEY=your_key
```

## Backend `.env`

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret

CLOUDINARY_CLOUD_NAME=your_name
CLOUDINARY_API_KEY=your_key
CLOUDINARY_API_SECRET=your_secret

STRIPE_SECRET=your_secret

RAZORPAY_KEY_ID=your_id
RAZORPAY_SECRET=your_secret

OPENAI_API_KEY=your_key
```

---

# ▶️ Run Locally

## Start Backend


## Start Frontend

